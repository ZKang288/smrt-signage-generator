# smrt-signage-generator
Generating SMRT signages

Font taken from: github.com/jglim/IdentityFont?tab=readme-ov-file#there-seems-to-be-another-font-thats-used-in-our-public-transport

These are not exactly to LTA specifications, but the maps and caplet sizes are adjustable to be as close to LTA's style as much as possible
